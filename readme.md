# Canteen Management Tool

A comprehensive solution for managing canteen  operations efficiently.

## Features

- Order management
- Inventory tracking
- Menu management
- User authentication
- Payment processing.

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

## Configuration

Create a `.env` file in the root directory with the required environment variables.
